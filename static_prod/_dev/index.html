How do I use Ugnis?

You use the visual editor to create the definition of your application. Once your app is ready, you can export a minified JavaScript code and put it in your website.

How do I save my application?

Ugnis is changing rapidly and we cannot support saving applications before we release 1.0v

How does Ugnis work internally?

Ugnis is declarative programming taken to the extreme. There are 4 parts:

  1. Your application as a JSON file
  2. Interpreter for your JSON code
  3. Editor, because you wouldn't want to write JSON by hand. Editor works with the Interpreter to change your code definition JSON.
  4. Compiler that takes your JSON and makes it into an optimized JS code that does not require an interpreter.

Will ugnis do ...?

Check out our road map. If there is something we haven't thought about, please send us an email or tweet

Can Ugnis create an app for Windows, Linux and Mac?

Yes, Ugnis 1.0v will allow automatic wrapping with Electron.

Can Ugnis create and iOS or Android app?

Not yet, though in the future a slightly changed editor with more specialized components will be able to export to iOS and Android.

Visual programming will never work.

But it already works. Ugnis did not start as a visual tool. It became obvious that we needed more than one dimension to express the cyclical nature of applications.
